파이썬 문제 - 두 수 비교

문제 설명

정수 num1과 num2가 입력됩니다.
두 수가 같으면 1, 다르면 -1을 출력해주세요.

제한사항

  • 0 ≤ num1 ≤ 10,000
  • 0 ≤ num2 ≤ 10,000
  • input() 함수 안에는 아무 문자열도 넣지 마세요.

입출력 예

번호num1num2결과
1"2""3"-1
2"11""11"1
3"7""99"-1

출력 예시 설명

예제 1: 2와 3은 다르므로 -1 출력

예제 2: 11과 11은 같으므로 1 출력

예제 3: 7과 99는 다르므로 -1 출력
