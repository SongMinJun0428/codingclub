파이썬 문제 - 두 정수의 곱

문제 설명

정수가 두 번 입력될 때, 두 정수의 곱을 출력해주세요.

제한사항

  • 0 ≤ 정수1 ≤ 100
  • 0 ≤ 정수2 ≤ 100
  • input() 함수 안에는 어떤 문자열도 쓰지 마세요.

입출력 예

번호[정수1][정수2]결과
1"2""3"6
2"100""2"200

출력 예시 설명

예제 1: 2 × 3 = 6 출력

예제 2: 100 × 2 = 200 출력

예상 코드 예시

a = int(input())
b = int(input())
print(a * b)
