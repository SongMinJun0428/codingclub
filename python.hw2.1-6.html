파이썬 문제 - 출생 연도 계산

문제 설명

철수는 40살인 선생님이 몇 년도에 태어났는지 궁금해졌습니다. 나이 [정수]가 입력될 때, 2022년을 기준으로 출생 연도를 출력해주세요.

제한사항

  • 0 < [정수] ≤ 120
  • 나이는 태어난 연도에 1살이며 1년마다 1씩 증가합니다.
  • input() 함수 안에는 문자열을 쓰지 마세요.

입출력 예

번호[정수]결과
1"40"1983
2"23"2000

출력 예시 설명

예제 1: 2022년 기준 40살 → 2022 - 40 + 1 = 1983

예제 2: 2022년 기준 23살 → 2022 - 23 + 1 = 2000

예상 코드 예시

age = int(input())
birth_year = 2022 - age + 1
print(birth_year)
