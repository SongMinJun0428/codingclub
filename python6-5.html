07-05. elif 사용하기

elif는 if 조건이 False일 때 실행될 수 있는 추가 조건을 처리할 때 사용됩니다. 중첩된 if-else 구조를 깔끔하게 바꿀 수 있습니다.

기본 예제: 양수/음수/0 판별

a = int(input())
if a > 0:
    print('양수입니다.')
elif a < 0:
    print('음수입니다.')
else:
    print('0입니다.')

입력 값에 따라 하나의 메시지가 출력됩니다.

여러 개의 elif 사용

a = int(input())
if a >= 90:
    print('A')
elif a >= 80:
    print('B')
elif a >= 70:
    print('C')
elif a >= 60:
    print('D')
elif a >= 50:
    print('E')
else:
    print('F')

점수에 따라 A~F 등급이 출력됩니다.

중첩된 if → elif로 개선

def checkValid(st):
    return st in ('가위', '바위', '보')

def isAWin(A, B):
    return (A, B) in (
        ('가위', '보'),
        ('바위', '가위'),
        ('보', '바위')
    )

A = input('A가 낸 것 : ')
B = input('B가 낸 것 : ')

if not (checkValid(A) and checkValid(B)):
    print('오류')
elif A == B:
    print('비겼습니다')
elif isAWin(A, B):
    print('A가 이겼습니다.')
else:
    print('B가 이겼습니다.')

가위바위보 게임 결과에 따라 결과가 출력됩니다.

불필요한 중첩 없이 조건을 elif로 명확히 구분하여 가독성을 높인 예제입니다.
