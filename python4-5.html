파이썬 기초 - 함수 예제 모음

함수 연습을 통해 다양한 조건 분기, 수학 연산, 문자열 처리 능력을 향상시켜 보세요.

짝수이면 True, 홀수이면 False 반환

def solution(num: int):
    return num % 2 == 0

짝수면 0, 홀수면 1 반환

def solution(num: int):
    return num % 2

짝수면 "even", 홀수면 "odd" 반환

def solution(num: int):
    return "even" if num % 2 == 0 else "odd"

문자가 모음이면 'a', 자음이면 'b'

def solution(cha: str):
    return 'a' if cha in 'aeiou' else 'b'

문자가 대문자면 True, 아니면 False

def solution(cha: str):
    return cha.isupper()

점수가 70점 이상이면 "Pass" 아니면 False

def solution(score: int):
    return "Pass" if score >= 70 else False

맞은 개수에 따라 점수 계산

def solution(score: int):
    if score <= 2:
        return 0
    elif score <= 5:
        return 3
    elif score <= 8:
        return 6
    else:
        return 9

약수인지 확인

def solution(m: int, n: int):
    return n % m == 0

1부터 3씩 증가하는 수열인지 확인

def solution(num: int):
    return (num - 1) % 3 == 0

3의 배수 또는 4의 배수인지 확인

def solution(num: int):
    return num % 3 == 0 or num % 4 == 0

사과를 상자에 담기 (8개 기준)

def solution(apple: int):
    return (apple + 7) // 8

사과와 상자 크기 기준 상자 개수

def solution(apple: int, box: int):
    return (apple + box - 1) // box

사과 구매 최소 금액 계산

def solution(apple: int):
    box = apple // 10
    single = apple % 10
    return box * 9500 + single * 1000

은행 이자 계산 (매년 두 배)

def solution(year: int):
    return 10000 * (2 ** year)

동전 최소 개수로 거스름돈 만들기

def solution(money: int):
    count = 0
    for coin in [500, 50, 10]:
        count += money // coin
        money %= coin
    return count

학생 이름과 상장 메시지 출력

def solution(name: str, award: str):
    return f"{name} 학생은 1학기에 다양한 활동으로\n{award}를 받았습니다. 축하드립니다."

보충 대상 여부 메시지 출력

def solution(name: str, score: int):
    if score < 70:
        return f"{name} 학생은 이번주 테스트에서\n{score}점을 받았으므로\n보충 대상이 맞습니다."
    else:
        return f"{name} 학생은 이번주 테스트에서\n{score}점을 받았으므로\n보충 대상이 아닙니다."
