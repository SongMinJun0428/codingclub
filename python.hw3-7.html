파이썬 문제 - 미성년자 판별

문제 설명

철수는 편의점 아르바이트를 하고 있습니다.
담배 구매자의 생년월일 year, month, date가 입력될 때,

2023년 3월 1일 기준으로 만 18세 이상이면 True, 아니면 False를 출력하세요.

제한사항

  • 1900 < year < 2023
  • 1 ≤ month ≤ 12
  • 1 ≤ date ≤ 31
  • 날짜 형식은 실제 달력에 맞게 입력됨을 가정합니다.
  • input() 함수 안에는 아무 문자열도 넣지 마세요.

입출력 예

번호yearmonthdate결과
1"2005""5""3"False
2"2004""12""30"True
3"1920""1""1"True

출력 예시 설명

예제 1: 2005년 5월 생 → 아직 생일 안 지났음 → 만 17세 → False 출력

예제 2: 2004년생은 이미 만 18세 → True 출력

예제 3: 1920년생은 당연히 성인 → True 출력
