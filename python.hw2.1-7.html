파이썬 문제 - 1000을 곱하고 소수 버리기

문제 설명

정수가 두 번 입력될 때, [정수1]을 [정수2]로 나눈 값에 1000을 곱한 뒤 소수 부분을 버린 정수를 출력해주세요.

제한사항

  • 0 ≤ [정수1] ≤ 100
  • 0 ≤ [정수2] ≤ 100
  • 단, input() 함수 안에는 문자열을 쓰지 마세요.

입출력 예

번호[정수1][정수2]결과
1"3""2"1500
2"7""3"2333
3"1""16"62

출력 예시 설명

  • 예제 1: 3 ÷ 2 = 1.5 → 1.5 × 1000 = 1500 → 1500
  • 예제 2: 7 ÷ 3 ≈ 2.333 → 2.333 × 1000 = 2333.3 → 2333
  • 예제 3: 1 ÷ 16 = 0.0625 → 0.0625 × 1000 = 62.5 → 62

예상 코드 예시

a = int(input())
b = int(input())
result = (a / b) * 1000
print(int(result))
