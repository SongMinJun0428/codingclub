07-04. else와 if 중첩

가위바위보 문제를 통해 if, else, 중첩 조건문, 함수, 튜플을 함께 활용하는 방법을 배워봅니다.

입력 값 유효성 검사

def checkValid(st):
    return st in ('가위', '바위', '보')

"가위", "바위", "보" 중 하나인지 확인

승리 판별 함수

def isAWin(A, B):
    return (A, B) in (
        ('가위', '보'),
        ('바위', '가위'),
        ('보', '바위')
    )

A가 이기는 경우를 튜플로 표현하여 간결하게 구현

가위바위보 전체 코드

def checkValid(st):
    return st in ('가위', '바위', '보')

def isAWin(A, B):
    return (A, B) in (
        ('가위', '보'),
        ('바위', '가위'),
        ('보', '바위')
    )

A = input('A가 낸 것 : ')
B = input('B가 낸 것 : ')

if checkValid(A) and checkValid(B):
    if A == B:
        print('비겼습니다')
    else:
        if isAWin(A, B):
            print('A가 이겼습니다.')
        else:
            print('B가 이겼습니다.')
else:
    print('오류')

입력된 값에 따라 A, B 중 승자가 결정되거나 비김, 오류 메시지가 출력됩니다.
