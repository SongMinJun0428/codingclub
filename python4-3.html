함수의 매개변수

def whoami(name, age):
    print("이름 :", name, "\n나이 :", age)

whoami("인코딩", 10)

이름 : 인코딩
나이 : 10

덧셈 함수

def add(a, b):
    return a + b

print(add(10, 13))

23

print(add(add(10, 32), add(10, add(-100, 30)))

-18

순수 함수와 비순수 함수

비순수 함수 예시

n = 0
def countUp():
    global n
    n += 1
    return n

print(countUp())
print(countUp())
print(countUp())

1
2
3

순수 함수로 변경

def countUp(n):
    return n + 1

n = countUp(0)
print(n)
n = countUp(n)
print(n)
n = countUp(n)
print(n)

1
2
3
