06-03. 튜플 사용해보기

튜플은 리스트와 비슷하지만 수정이 불가능한 자료형입니다. 불변(immutable) 객체라는 점이 가장 큰 특징입니다.

튜플 생성

a = (1, 2, 3)
print(a)

(1, 2, 3)

()는 생략 가능하며, 이 경우에도 튜플로 인식됩니다:

a = 1, 2, 3
print(a)

(1, 2, 3)

다중 할당 예제:

x = 1, 2, 3
x1, x2, x3 = x

print(x1)
print(x2)
print(x3)

1
2
3

다양한 타입도 저장할 수 있습니다:

a = ("인코딩", 3, "플러스", 5)
print(a)

('인코딩', 3, '플러스', 5)

주의! 하나짜리 튜플은 쉼표(,)가 필요합니다:

a = (1)
b = (1,)

print(a, type(a))
print(b, type(b))

1 <class 'int'>
(1,) <class 'tuple'>

튜플의 연산

튜플도 + 와 * 연산이 가능합니다:

a = (1, 2, 3)
print(a + (4, 5, 6))
print(a * 3)

(1, 2, 3, 4, 5, 6)
(1, 2, 3, 1, 2, 3, 1, 2, 3)

하지만 이것은 튜플 자체를 변경하는 것이 아니라, 새로운 튜플을 생성한 것입니다:

a = (1, 2, 3)
a = a + (4, 5, 6)
print(a)

(1, 2, 3, 4, 5, 6)

※ 튜플은 내부 요소의 변경은 불가하지만, 재할당은 가능합니다.

튜플과 리스트 타입 변환

a = (1, 2, 3)
print(a, type(a))

a = list(a)
print(a, type(a))

a = tuple(a)
print(a, type(a))

(1, 2, 3) <class 'tuple'>
[1, 2, 3] <class 'list'>
(1, 2, 3) <class 'tuple'>

타입 비교도 가능합니다:

a = [1, 2, 3]
print(type(a) is tuple)
print(type(a) is list)

a = tuple(a)
print(type(a) is tuple)
print(type(a) is list)

False
True
True
False
