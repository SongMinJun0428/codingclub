06-03. 튜플 사용해보기

튜플이란?

튜플은 리스트와 비슷하지만 값을 변경할 수 없는 자료형입니다.

a = (1, 2, 3)
print(a)
(1, 2, 3)

괄호 () 없이도 튜플을 만들 수 있습니다.

a = 1, 2, 3
print(a)
(1, 2, 3)

변수 다중 할당은 사실상 튜플입니다.

x = 1, 2, 3
x1, x2, x3 = x
print(x1)
print(x2)
print(x3)
1
2
3

다양한 타입의 값도 가능

a = ("인코딩", 3, "플러스", 5)
print(a)
('인코딩', 3, '플러스', 5)

하나짜리 튜플 주의

a = (1)
b = (1,)
print(a, type(a))
print(b, type(b))
1 <class 'int'>
(1,) <class 'tuple'>
주의: 하나의 값만 있는 튜플은 꼭 ,를 써야 튜플로 인식됩니다.

튜플 연산

a = (1, 2, 3)
print(a + (4, 5, 6))
print(a * 3)
(1, 2, 3, 4, 5, 6)
(1, 2, 3, 1, 2, 3, 1, 2, 3)

튜플은 +와 * 연산이 가능하지만, 값 자체를 바꾸는 건 불가능합니다.

a = (1, 2, 3)
a = a + (4, 5, 6)
print(a)
(1, 2, 3, 4, 5, 6)
재할당은 가능하지만, 튜플 안의 값을 직접 수정하는 것은 불가능합니다.

튜플 ↔ 리스트 변환

a = (1, 2, 3)
print(a, type(a))

a = list(a)
print(a, type(a))

a = tuple(a)
print(a, type(a))
(1, 2, 3) <class 'tuple'>
[1, 2, 3] <class 'list'>
(1, 2, 3) <class 'tuple'>

타입 비교

a = [1, 2, 3]
print(type(a) is tuple)
print(type(a) is list)

a = tuple(a)
print(type(a) is tuple)
print(type(a) is list)
False
True
True
False
