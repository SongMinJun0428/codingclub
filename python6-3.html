07-03. else 사용하기

else는 if의 조건이 False일 때 실행되는 코드를 작성할 수 있도록 도와줍니다.

기존 방식 (if 두 번)

age = 18
if age < 18:
    print('미성년자입니다.')
if age >= 18:
    print('성인입니다.')

성인입니다.

두 개의 조건문이 모두 검사되기 때문에 비효율적입니다.

논리 연산자를 활용한 if

age = 18
if age < 18:
    print('미성년자입니다.')
if not (age < 18):
    print('성인입니다.')

성인입니다.

논리 연산자를 써서 조건을 반전시킬 수 있지만 가독성이 떨어질 수 있습니다.

else 문법 사용

a = 3
if a == 3:
    print('3입니다.')
else:
    print('3이 아닙니다.')

3입니다.

a = 4
if a == 3:
    print('3입니다.')
else:
    print('3이 아닙니다.')

3이 아닙니다.

미성년자 판별 코드 개선

age = 18
if age < 18:
    print('미성년자입니다.')
else:
    print('성인입니다.')

성인입니다.

else를 사용하면 불필요한 조건 평가 없이 더 간결하고 빠른 코드를 작성할 수 있습니다.
