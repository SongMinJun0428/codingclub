파이썬 문제 - 평균 속력 계산

문제 설명

철수는 9.81km를 갈 때는 13.7km/h의 속도로,
돌아올 때는 4.8km/h의 속도로 이동했습니다.
이때 왕복 평균 속력을 x km/h라고 할 때,
100x의 정수 부분을 출력하는 프로그램을 작성하세요.

제한사항

  • 출력은 반드시 정수만 출력하며, % 기호나 다른 문자는 포함하지 않습니다.
  • 정답이 100x = 583.33이라면 583으로 출력되어야 합니다.

입출력 예

입력

없음

출력

정수 결과만 한 줄로 출력합니다. 

583
