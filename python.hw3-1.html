파이썬 문제 - 짝수 판별

문제 설명

정수가 입력될 때, 그 정수가 짝수이면 True, 홀수이면 False를 출력하세요.

C언어로 문제를 푸는 학생은 1(짝수), 0(홀수)으로 출력되게 코딩을 완성하세요.

제한사항

  • 0 < [정수] < 10,000
  • input() 함수 안에는 아무 문자열도 넣지 마세요.

입출력 예

번호[정수]결과
1"320"True
2"201"False
3"800"True

출력 예시 설명

예제 1: 320은 짝수이므로 True 출력

예제 2: 201은 홀수이므로 False 출력

예제 3: 800은 짝수이므로 True 출력
