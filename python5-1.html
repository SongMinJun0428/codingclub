06-01. 문자열 사용해보기

문자열은 파이썬의 기본 값 중 하나입니다. 문장을 출력할 때에도 문자열을 사용합니다.

print('Hello World')

Hello World

작은 따옴표(') 또는 큰 따옴표(")로 문자열을 감쌀 수 있지만, 혼용하면 오류가 발생합니다.

print('Hello World")

SyntaxError: unterminated string literal

문자열 연산

a = 'Hello'
b = 'World'
print(a + b)

HelloWorld

a = 'Hello'
b = 'World'
print(a + ' ' + b)

Hello World

a = 'aaa'
b = 'bbb'
print(a + b)
print(b + a)

aaabbb
bbbaaa

숫자와 문자열을 더하면 오류가 납니다:

a = 1000
b = 'Hello'
print(a + b)

TypeError: unsupported operand type(s) for +: 'int' and 'str'

문자열로 변환 후 더해야 합니다:

a = 1000
b = 'Hello'
print(str(a) + b)

1000Hello

* 연산

a = 'hell'
print(a * 5)

hellhellhellhellhell

a = 'hell'
b = '5'
print(a * b)

TypeError: can't multiply sequence by non-int of type 'str'

a = 'hell'
b = 5.0
print(a * b)

TypeError: can't multiply sequence by non-int of type 'float'

정수와의 곱셈은 순서가 바뀌어도 가능합니다:

a = 'hell'
b = 5
print(b * a)
print(a * b)

hellhellhellhellhell
hellhellhellhellhell

문자열 변환

a = str(30)
print(a)

b = str(10.4)
print(b)

c = str(True)
print(c)

30
10.4
True

문자열 결합 후 숫자로 변환:

a = 30
b = 1
c = str(a) + str(b)
c = int(c)
print(c)

301

가끔 하는 착각

a = 'hell'
a * 5
print(a)

hell

※ 연산 결과를 저장하지 않으면 원래 값은 변하지 않습니다.

a = 'hell'
a = a * 5
print(a)

hellhellhellhellhell
