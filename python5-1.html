06-01. 문자열 사용해보기

문자열 기본

문자열은 작은 따옴표(')나 큰 따옴표(")로 감싸서 만듭니다.

print('Hello World')
Hello World
print('Hello World")
SyntaxError: unterminated string literal

문자열은 같은 종류의 따옴표로 시작하고 끝나야 합니다.

문자열 연산

문자열은 + 와 * 연산을 지원합니다.

a = 'Hello'
b = 'World'
print(a + b)
HelloWorld
print(a + ' ' + b)
Hello World
a = 'aaa'
b = 'bbb'
print(a + b)
print(b + a)
aaabbb
bbbaaa
a = 1000
b = 'Hello'
print(a + b)
TypeError: unsupported operand type(s) for +: 'int' and 'str'
print(str(a) + b)
1000Hello

* 연산

a = 'hell'
print(a * 5)
hellhellhellhellhell
a = 'hell'
b = '5'
print(a * b)
TypeError: can't multiply sequence by non-int of type 'str'
b = 5.0
print(a * b)
TypeError: can't multiply sequence by non-int of type 'float'
b = 5
print(b * a)
hellhellhellhellhell

문자열 변환

a = str(30)
b = str(10.4)
c = str(True)
print(a)
print(b)
print(c)
30
10.4
True

문자열 변환은 숫자를 문자열로 바꿔 연결할 때 자주 사용됩니다.

a = 30
b = 1
c = str(a) + str(b)
c = int(c)
print(c)
301

가끔 하는 착각

a = 'hell'
a * 5
print(a)
hell

주의: 연산만 하면 값이 바뀌지 않습니다. 반드시 다시 저장해야 합니다.

a = 'hell'
a = a * 5
print(a)
hellhellhellhellhell
