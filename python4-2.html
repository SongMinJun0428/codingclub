파이썬 기초 - 함수의 반환

함수는 단순히 코드를 실행하는 것뿐만 아니라 호출 후 어떤 값을 반환할 수 있습니다.

기본적인 반환

def whoami():
    pass

print(whoami())

None

아무것도 반환하지 않으면 None이 반환됩니다.

print vs return

def whoami():
    print('Incoding')

name = whoami()
print(name)

Incoding
None

print는 화면에 출력만 하고 값을 반환하지 않습니다.

def whoami():
    return 'Incoding'

name = whoami()
print(name)

Incoding

return은 값을 반환하며 함수 실행을 종료합니다.

return 이후 코드

def whoami():
    print('show')
    return 'Incoding'
    print('now show')

name = whoami()
print(name)

show
Incoding

return 이후의 코드는 실행되지 않습니다.

예제: 변수를 누적 할당하자

d = 0
def counter():
    global d
    d += 1
    return d

a = counter()
b = counter()
c = counter()
print(a, b, c)

1 2 3

함수 호출마다 d의 값이 증가하며 return을 통해 반환됩니다.

  • global 키워드 없이 d를 수정하려 하면 오류가 발생합니다.
  • a, b, c는 각각 호출 시점의 d 값을 복사해서 갖습니다.
