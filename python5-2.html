06-02. 리스트 사용해보기

리스트란?

여러 개의 데이터를 하나의 변수에 담고 싶을 때, 리스트(list)를 사용합니다.

a = 10
b = 5.3214
c = 'abc'
d = True

예를 들어 1학년 1반 학생 목록을 각각의 변수로 저장하면 다음과 같습니다.

student1 = "인코딩"
student2 = "플러스"
student3 = "파이썬"

이런 방식은 관리가 어렵기 때문에 리스트를 사용하는 것이 좋습니다.

리스트 생성

student = ["인코딩", "플러스", "파이썬"]
print(student)
['인코딩', '플러스', '파이썬']

리스트 안에는 서로 다른 타입의 값도 넣을 수 있습니다.

a = 10
student = ["인코딩", 3, "플러스", a]
print(student)
['인코딩', 3, '플러스', 10]

리스트 연산

리스트는 문자열처럼 + 와 * 연산이 가능합니다.

a = [1, 2, 3]
b = [4, 5, 6]
print(a + b)
[1, 2, 3, 4, 5, 6]

주의: 단순히 더한다고 해서 a나 b의 값은 바뀌지 않습니다.

a = [1, 2, 3]
b = [4, 5, 6]
a = a + b
print(a)

a += b
print(a)
[1, 2, 3, 4, 5, 6]
[1, 2, 3, 4, 5, 6, 4, 5, 6]

* 연산

리스트는 정수와 곱하면 반복됩니다.

a = [1, 2]
print(a * 5)
[1, 2, 1, 2, 1, 2, 1, 2, 1, 2]
print(a)
[1, 2]
a = 5 * a
print(a)
[1, 2, 1, 2, 1, 2, 1, 2, 1, 2]
참고: 문자열과 마찬가지로 리스트도 재할당하지 않으면 원래 변수는 바뀌지 않습니다.
