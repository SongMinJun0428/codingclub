06-02. 리스트 사용해보기

지금까지는 하나의 변수에 하나의 값만 저장했지만, 리스트를 사용하면 여러 값을 하나의 변수에 저장할 수 있습니다.

a = 10
b = 5.3214
c = 'abc'
d = True

예를 들어, 1학년 1반 학생 목록을 저장할 때 리스트를 쓰는 것이 더 효율적입니다:

student = ["인코딩", "플러스", "파이썬"]
print(student)

['인코딩', '플러스', '파이썬']

리스트에는 다양한 타입의 데이터를 함께 넣을 수도 있습니다:

a = 10
student = ["인코딩", 3, "플러스", a]
print(student)

['인코딩', 3, '플러스', 10]

리스트의 연산

a = [1, 2, 3]
b = [4, 5, 6]
print(a + b)

[1, 2, 3, 4, 5, 6]

※ 연산만 하면 원본 리스트는 바뀌지 않습니다.

a = [1, 2, 3]
b = [4, 5, 6]

a = a + b
print(a)

a += b
print(a)

[1, 2, 3, 4, 5, 6]
[1, 2, 3, 4, 5, 6, 4, 5, 6]

* 연산

a = [1, 2]
print(a * 5)
print(a)
a = 5 * a
print(a)

[1, 2, 1, 2, 1, 2, 1, 2, 1, 2]
[1, 2]
[1, 2, 1, 2, 1, 2, 1, 2, 1, 2]
