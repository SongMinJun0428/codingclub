파이썬 기초 - 함수를 문제 푸는 데 사용하기

숙제를 제출할 때는 solution이라는 이름의 함수를 사용해 제출합니다. 문제 예제를 통해 배워봅시다.

예제: 두 수의 덧셈

문제 설명: 정수 num1과 num2가 주어질 때, num1과 num2의 합을 return하도록 solution 함수를 완성해주세요.

def solution(num1, num2):
    return num1 + num2

입출력 예: solution(2, 3) → 5
solution(100, 2) → 102

함수 말고 다른 곳에 코드를?

def add(a, b):
    return a + b

def solution(num1, num2):
    return add(num1, num2)

외부 함수를 활용해도 무방합니다. 하지만 solution 함수의 이름은 바꾸면 안 됩니다.

매개변수 이름을 바꿔도?

def solution(a, b):
    x = a + b
    return x

매개변수 이름은 바꿔도 상관없고, 함수 내부 코드를 자유롭게 작성해도 됩니다.

테스트는 어떻게?

def solution(num1, num2):
    return num1 - num2

print(solution(10, 20) == 30)
print(solution(0, 0) == 0)
print(solution(1000, 2000) == 3000)

False
True
False

테스트를 직접 해보며 예측값과 비교해 오답을 잡아냅니다.

def solution(num1, num2):
    return num1 + num2

print(solution(10, 20) == 30)
print(solution(0, 0) == 0)
print(solution(1000, 2000) == 3000)

True
True
True

이제 모든 테스트가 통과된 것을 확인할 수 있습니다. 다양한 테스트 케이스를 넣어보는 습관을 들이세요.
