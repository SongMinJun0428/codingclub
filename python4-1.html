파이썬 기초 - 함수의 호출

프로그래밍 언어는 반복적인 작업을 대신하기 위해 만들어졌습니다. 그 속성 중 하나가 바로 함수입니다.

함수에 있는 print

def printSomething():
    print('Incoding')

printSomething()

Incoding

함수를 호출하려면 함수 이름 옆에 ()를 써야 합니다.

def printMultiple():
    print('hello')
    print('end')
    print('good bye')

printMultiple()
print('----------')
printMultiple()

hello
end
good bye
----------
hello
end
good bye

함수는 여러 번 호출할 수 있습니다.

pass

def notNow():
    pass

print('Some Code')

Some Code

pass는 나중에 코드를 채우기 위해 임시로 사용하는 키워드입니다.

함수의 위치

notWork()
def notWork():
    print('not show')

NameError: name 'notWork' is not defined

함수는 반드시 호출 전에 정의되어 있어야 합니다.

def notWork():
    print('now show')

notWork()

now show

global

a = 20
def makeFun():
    a = 30

makeFun()
print(a)

20

함수 안의 a는 외부의 a와 다릅니다.

a = 20
def makeFun():
    global a
    a = 30

makeFun()
print(a)

30

global 키워드를 사용하면 외부 변수에 접근할 수 있습니다.

a = 20
def makeFun():
    print(a)

makeFun()

20

읽기만 할 경우엔 global이 없어도 됩니다.
