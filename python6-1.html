07-01. if 사용하기

이제 조건에 따라 다른 동작을 하는 분기를 배워봅니다. 예를 들어, 나이에 따라 출력 내용을 다르게 하는 조건문을 만들 수 있습니다.

if 문법

a = 3
if a == 3:
    print(a)

3

조건이 True일 경우에만 실행됩니다.

if에서 들여쓰기

a = 3
if a == 3:
    print(f'여긴 if 안쪽 {a}')
print(f'여긴 if 밖 {a}')

여긴 if 안쪽 3
여긴 if 밖 3

a = 4
if a == 3:
    print(f'여긴 if 안쪽 {a}')
print(f'여긴 if 밖 {a}')

여긴 if 밖 4

if에서 여러 줄

a = 4
if a == 4:
    print('a')
    print('b')
    print('c')
print('x')

if에서 pass 사용

a = 3
if a == 3:
    pass  # 아무 동작도 하지 않음
print('계속 진행')

계속 진행

양수/음수/0 판별기

num = float(input('숫자를 입력해주세요 : '))

if num > 0:
    print(str(num) + '은/는 양수입니다.')
if num < 0:
    print(str(num) + '은/는 음수입니다.')
if num == 0:
    print('0입니다.')

입력한 수에 따라 결과 출력:
예: -2 → -2.0은/는 음수입니다.
