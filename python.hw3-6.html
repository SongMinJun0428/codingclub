파이썬 문제 - 윤년 판별

문제 설명

정수가 입력될 때, 해당 년도가 윤년인지 판별하세요.

윤년 조건은 아래와 같습니다:

  • 해당 년도가 400으로 나눠지면 윤년
  • 100으로 나눠지지 않고, 4로 나눠지면 윤년

예: 320년 → 윤년, 200년 → 윤년 아님, 800년 → 윤년

C언어로 문제를 푸는 학생은 1(참), 0(거짓)으로 출력되게 코딩을 완성하세요.

제한사항

  • 0 < [정수] < 10,000
  • input() 함수 안에는 아무 문자열도 넣지 마세요.

입출력 예

번호[정수]결과
1"320"True
2"200"False
3"800"True

출력 예시 설명

예제 1: 320은 4로 나눠지고 100으로 나눠지지 않으므로 윤년 → True

예제 2: 200은 100으로 나눠지지만 400으로는 나눠지지 않음 → False

예제 3: 800은 400으로 나눠짐 → 윤년 → True
