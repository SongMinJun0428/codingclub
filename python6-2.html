07-02. if 활용하기

이전의 미성년자 판별 문제를 함수로 바꾸어 재사용성과 가독성을 높여 봅시다.

함수 만들기 (기초)

def checkChild(age):
    if age < 18:
        print('미성년자입니다.')
    if age >= 18:
        print('성인입니다.')

checkChild(17)
checkChild(18)
checkChild(19)

미성년자입니다.
성인입니다.
성인입니다.

※ age < 18의 반대 조건을 정확하게 표현하려면 age >= 18이 더 명확합니다.

return을 사용한 간결한 분기

def checkChild(age):
    if age < 18:
        print('미성년자입니다.')
        return
    print('성인입니다.')

checkChild(17)
checkChild(18)
checkChild(19)

미성년자입니다.
성인입니다.
성인입니다.

※ return을 이용해 미성년자일 경우 바로 함수를 종료합니다.

출력 분리를 위한 조건 반환 함수

def checkChild(age):
    return age < 18

if checkChild(17):
    print('미성년자입니다.')
if not checkChild(17):
    print('성인입니다.')

미성년자입니다.

※ 함수의 결과를 조건으로 활용하여 True/False에 따라 다른 출력을 분리합니다.
