06-05. 시퀀스 자료형의 기본

시퀀스는 연속된 값을 저장하는 자료형입니다. 파이썬에서 대표적인 시퀀스는 다음과 같습니다:

  • list : 리스트
  • tuple : 튜플
  • str : 문자열
  • range : 연속된 숫자 생성

in 연산자

시퀀스에 특정 값이 포함되어 있는지를 확인할 수 있습니다.

a = [1, 5, 9]
b = (1, 5, 9)
c = '159'
d = range(1, 10, 4)

print(5 in a, 5 in b, '5' in c, 5 in d)
print(4 in a, 4 in b, '4' in c, 4 in d)

True True True True
False False False False

부분 문자열 검색은 불가능합니다:

a = ('kim', 'han', 'kel')
b = ['kim', 'han', 'kel']

print('ki' in a, 'ki' in b)
print('kim' in a, 'kim' in b)

False False
True True

len 함수

시퀀스의 길이(항목 수)를 반환합니다.

a = [1, 5, 9]
b = (1, 5, 9)
c = '159'
d = range(1, 10, 4)

print(len(a), len(b), len(c), len(d))

3 3 3 3

리스트나 튜플에 문자열이 들어 있어도 항목 수 기준입니다:

a = ('kim', 'han', 'kel')
b = ['kim', 'han', 'kel']
print(len(a), len(b))

3 3

문자열의 길이는 문자 수 기준입니다 (공백, 줄바꿈 포함):

a = 'abc def'
b = 'abc\n\n\ndef\n\n\nghi'
print(len(a), len(b))

7 15

인덱싱 (Indexing)

시퀀스에서 특정 위치의 값을 가져올 수 있습니다. 인덱스는 0부터 시작합니다.

a = [1, 5, 9]
b = (1, 5, 9)
c = '159'
d = range(1, 10, 4)

print(a[0])
print(b[1])
print(c[2])
print(d[1])

1
5
9
5

음수 인덱스를 사용하면 뒤에서부터 접근할 수 있습니다:

print(a[-1])
print(b[-2])
print(c[-3])
print(d[-1])

9
5
1
9

※ 범위를 벗어나면 IndexError가 발생합니다.

a = [1, 2, 3]
print(a[3])
print(a[-4])

IndexError: list index out of range
