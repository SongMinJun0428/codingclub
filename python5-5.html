06-05. 시퀀스 자료형의 기본

시퀀스란?

파이썬의 시퀀스(sequence) 자료형은 연속된 값을 가진 자료형을 말하며, 대표적으로 다음이 있습니다:

  • list : 리스트
  • tuple : 튜플
  • str : 문자열
  • range : 숫자 범위 생성

in 연산자

값이 시퀀스에 포함되어 있는지 확인합니다.

a = [1, 5, 9]
b = (1, 5, 9)
c = '159'
d = range(1, 10, 4)

print(5 in a, 5 in b, '5' in c, 5 in d)
print(4 in a, 4 in b, '4' in c, 4 in d)
True True True True
False False False False
a = ('kim', 'han', 'kel')
b = ['kim', 'han', 'kel']

print('ki' in a, 'ki' in b)
print('kim' in a, 'kim' in b)
False False
True True

len 함수

시퀀스의 길이를 반환합니다.

a = [1, 5, 9]
b = (1, 5, 9)
c = '159'
d = range(1, 10, 4)

print(len(a), len(b), len(c), len(d))
3 3 3 3
a = ('kim', 'han', 'kel')
b = ['kim', 'han', 'kel']
print(len(a), len(b))
3 3
a = 'abc def'
b = 'abc\n\n\ndef\n\n\nghi'
print(len(a), len(b))
7 15

인덱스

시퀀스의 특정 위치 값을 가져올 때는 []를 사용합니다.

a = [1, 5, 9]
b = (1, 5, 9)
c = '159'
d = range(1, 10, 4)

print(a[0])
print(b[1])
print(c[2])
print(d[1])
1
5
9
5

음수 인덱스

print(a[-1])
print(b[-2])
print(c[-3])
print(d[-1])
9
5
1
9

잘못된 인덱스

a = [1, 2, 3]

print(a[3])
print(a[-4])
IndexError: list index out of range
IndexError: list index out of range
인덱스는 0부터 시작하며, 범위를 벗어나면 IndexError가 발생합니다.
