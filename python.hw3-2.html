파이썬 문제 - 배수 판별

문제 설명

정수 m과 n이 각각 입력될 때,
m이 n의 배수이면 True를, 배수가 아니라면 False를 출력해주세요.

C언어로 문제를 푸는 학생은 1(참), 0(거짓)으로 출력되게 코딩을 완성하세요.

제한사항

  • 2 ≤ m ≤ 100
  • 2 ≤ n ≤ 9
  • input() 함수 안에는 아무 문자열도 넣지 마세요.

입출력 예

번호mn결과
1"98""2"True
2"34""3"False

출력 예시 설명

예제 1: 98은 2의 배수이므로 True 출력

예제 2: 34는 3의 배수가 아니므로 False 출력
